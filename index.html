Welcome to the Resona Model Application

This project implements the advanced voice analysis, reproduction, and incubation methodologies developed in the Resona Model research. It provides a powerful toolset for understanding and working with brand and personal voice.

The full interactive application is built using Python (Flask) and requires a backend server to run. This static page serves as an introduction and provides access to the source code.

Running the Full Application

To use the interactive features (Master Analysis, Reproduction, Incubation), you need to run the Python application locally or deploy it to a hosting service that supports Python/Flask.

  1. Clone the repository from GitHub.
  2. Follow the setup instructions in the README.md file.
  3. Run the application using the provided `run.sh` script or manually with Flask.
  4. Access the application via your local browser (usually `http://localhost:5000`).

Note: This static page hosted on GitHub Pages does not include the interactive application functionality.
